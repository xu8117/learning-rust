### Rust 比 Golang 还难吗？

- Golang 入手简单，写完后你都不知道对错，Rust 入手难，写完后 90% 都是对的。
- 只学过 Rust 的一点点语法，晦涩难懂。

### Rust 的优势在哪里？

- 编译器很屌。
- 没有 gc，适合高性能低延时的服务。

### Rust 的应用场景是哪方面？

- TiKV 等
